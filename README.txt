Projecting the Career Trajectories of NFL Draft Prospects
Jack DeLaet, Connor Wolfe, Matthew Walter, Connor Blunt, Muhammad Ali Asjad

We have devised a model to project the career trajectories of defensive tackles for this upcoming NFL draft class. In the repository, you will find the csv files we used for our data as well as the code used to create the model and a Shiny App displaying the desired prospect's career trajectory.

To run the ML NFL Repository model:

Download all necessary csv files
1. Open githubpush.R
2. Run githubpush.R
3. Open Prospect Projections Shiny.R
4. Run Prospect Projections Shiny.R
5. View different prospect projection trajectories and enjoy!
